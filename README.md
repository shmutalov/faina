Faina is a [Faye](https://faye.jcoglan.com) implementation in Rust

# Building
```sh
cargo build
```

# Install

### Current version
```toml
[dependencies]
faina = "0.1.0"
```

### Development version
```toml
[dependencies]
faina = { git = "https://github.com/shmutalov/faina" }
```
